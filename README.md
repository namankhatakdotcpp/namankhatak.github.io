## 🧠 About This Portfolio

This is a fully custom-built interactive portfolio designed and engineered from scratch.

It includes:

- Scroll-driven canvas animation (41-frame sequence)
- Physics-based custom cursor
- Neural network-inspired loader animation
- Particle background system
- 3D interactive capability cards
- Dynamic metric counters with easing
- Skill filtering system
- Project filtering logic
- Resume download interaction
- GitHub stats integration
- Responsive glassmorphism UI

Built without frameworks — pure HTML, CSS, and JavaScript.
